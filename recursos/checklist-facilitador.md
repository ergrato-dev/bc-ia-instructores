# Checklist del facilitador

> Prepara todo esto **antes del día del curso**. No el mismo día.

---

## 📅 Una semana antes

- [ ] Confirmar número de participantes y sala asignada
- [ ] Verificar que la sala tiene proyector o pantalla + HDMI
- [ ] Verificar conexión a internet estable (probar velocidad)
- [ ] Crear cuentas de prueba en Claude y ChatGPT si no las tienes
- [ ] Activar la búsqueda web en Claude (herramientas > web search)
- [ ] Hacer un recorrido completo del repo `bc-ia-instructores` en GitHub
- [ ] Verificar que Syllabot está funcionando en syllabot.ergrato.dev
- [ ] Preparar capturas de pantalla de Syllabot como plan B
- [ ] Imprimir (o compartir digitalmente) el material del participante
- [ ] Probar todos los prompts de ejemplo en vivo — algunos pueden necesitar ajuste

---

## 📅 El día antes

- [ ] Cargar el computador completamente
- [ ] Abrir en pestañas del navegador (y no cerrar):
  - [ ] github.com/ergrato-dev (lista de repos bc-)
  - [ ] Un repo bc- de ejemplo para el tour
  - [ ] syllabot.ergrato.dev
  - [ ] claude.ai (sesión iniciada)
  - [ ] modelcontextprotocol.io
- [ ] Tener el prompt maestro del curso en un documento de texto para copiar rápido
- [ ] Preparar un ejemplo de output de IA bueno y uno malo para M2
- [ ] Revisar la plantilla de prompt SENA — estar listo para usarla en vivo

---

## ⏰ 30 minutos antes del curso

- [ ] Conectar el computador al proyector y verificar que se ve bien
- [ ] Ajustar resolución de pantalla si es necesario (1920x1080 recomendado)
- [ ] Probar volumen de parlantes si vas a mostrar algo con audio
- [ ] Abrir VS Code con el repo `bc-ia-instructores` clonado localmente
- [ ] Tener terminal abierta con Claude Code listo (opcional, para M4)
- [ ] Verificar conexión a internet una vez más
- [ ] Distribuir material impreso o compartir link del repo con participantes

---

## 🚨 Plan B para cada dependencia crítica

| Si falla... | Plan B |
|-------------|--------|
| Internet | Tener ejemplos de prompts y outputs ya guardados en el computador |
| Syllabot | Capturas de pantalla en `assets/syllabot-demo/` |
| Claude.ai | Usar ChatGPT o Gemini como alternativa |
| Proyector | Pasar el portátil por las mesas o describir verbalmente |
| Claude Code | Hacer la demo solo con la interfaz web de Claude |

---

## 📊 Estimación de materiales físicos

Si vas a imprimir:

| Documento | Páginas | Ejemplares |
|-----------|---------|------------|
| Material del participante (todos los módulos) | ~12 | 1 por persona + 2 extra |
| Plantilla de prompt SENA | 3 | 1 por persona |
| Glosario | 2 | 1 por persona |
| Taller T1 | 1 | 1 por persona |
| Taller T2 | 1 | 1 por pareja |

---

## ✅ Al final del curso

- [ ] Compartir el link del repo con todos los participantes
- [ ] Compartir link del canal @BCChannelEPTI
- [ ] Crear grupo de WhatsApp o canal de Teams para compartir prompts
- [ ] Recopilar compromisos de los participantes (foto del tablero si los escribieron ahí)
- [ ] Anotar qué funcionó bien y qué ajustar para la próxima vez
